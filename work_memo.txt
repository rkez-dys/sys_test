名前空間を使用して要素と属性をグループ化することができます。しかし、insertAdjacentHTMLは名前空間をサポートしていません。したがって、名前空間を持つXMLを作成することはできません。
&など文字化けする可能性がある
DOMツリーに追加するときはdocument.createElementを使用する必要がある

const parent = document.createElement("div");
const html = `
  <div>
    <p>Child Element 1</p>
    <p>Child Element 2</p>
  </div>
`;
parent.insertAdjacentHTML("beforeend", html);
document.body.appendChild(parent);
